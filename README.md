# Johor-Plantation-Palm-Performance-Dashboard
A palm production dashboard to analyze performance

An end-to-end Power BI analytics solution to monitor palm plantation operational performance. Built a complete data model covering production throughput, equipment efficiency, maintenance patterns, and energy consumption.
